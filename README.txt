Bookstore
---------

By Ash Laidlaw

Project Goals
-------------

This project is meant to teach Drupal's basic functionality, including using modules, blocks, pages, views, and more. It makes use of a few contributed modules as well as built-in modules. The distributed modules included are:
  • Views
  • Strongarm
  • Features
  • Chaos Tools

Setup
-----

In order to run this project, you will need the following programs installed on your computer:
  • MAMP or MAMP Pro
  • Terminal

To run the site, follow these instructions:
  1. Clone the project from https://github.com/Yhbv24/bookstore
  2. Open MAMP, and start the servers
  4. Navigate to http://localhost:8888/phpmyadmin/
  5. At the top of the screen, click "Import"
  6. Choose the zipped SQL file located in the project folder/sites/db-backup
  7. In MAMP, stop the servers, change the web server directory to the project folder, and start them again
  8. Navigate to http://localhost:8888/
  9. Log into the site with the username "bookstore" and the password "bookstore", which will give you administrative privileges
